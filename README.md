# Elastic Search AI Platform - Interactive Presentation

An interactive web presentation showcasing the Elastic Search AI Platform, built with React, Tailwind CSS, and Framer Motion.

## Features

- **8 Interactive Scenes** - Animated, engaging storytelling experience
- **Keyboard Navigation** - Use arrow keys, space, or number keys (1-8) to navigate
- **Smooth Transitions** - Spring-based page transitions with Framer Motion
- **Responsive Design** - Works on desktop and tablet screens
- **Interactive Elements** - Clickable cards, hover effects, animated counters

## Scenes

1. **Hero** - Dramatic introduction with floating data particles
2. **Data Explosion** - Interactive visualization of the 175ZB data challenge
3. **Challenges** - Tabbed exploration of Observability, Security, and Search challenges
4. **Platform** - Animated reveal of the Elastic platform pillars
5. **Business Value** - Interactive metrics dashboard with animated counters
6. **Unified Strategy** - Data flow visualization showing silo consolidation
7. **Team** - Contact cards for your Elastic team with copy-to-clipboard
8. **Next Steps** - Interactive journey timeline and CTA

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Navigation

- **Arrow Keys**: `←` Previous, `→` Next
- **Space / Enter**: Next scene
- **Number Keys**: Jump directly to scene 1-8
- **Dots**: Click the navigation dots on the right side
- **Arrows**: Click the arrow buttons on the sides

## Tech Stack

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)

## Customization

- **Colors**: Edit `tailwind.config.js` to modify the Elastic brand colors
- **Content**: Edit scene components in `src/scenes/` to update content
- **Animations**: Modify Framer Motion variants for custom transitions
